College Attendance Portal

A comprehensive attendance management system built with Java Swing and MySQL, featuring student tracking, report generation, and administrative controls.

Technologies & Software Used

Java Swing MySQL JDBC NetBeans / IDE

Why These Technologies Were Used

Java Swing

Used to build the graphical user interface (buttons, forms, windows). Java Swing provides a rich set of components for creating desktop applications with interactive elements like tables, forms, and dialog boxes.

MySQL

Used as the backend database to store student details and attendance records. MySQL provides reliable data storage, efficient querying, and data integrity for managing large amounts of student information.

JDBC

Used to connect Java Swing with the MySQL database. JDBC (Java Database Connectivity) acts as a bridge between the application and database, enabling data retrieval, insertion, and updates.

IDE (NetBeans / Eclipse)

Used for building, debugging, and running the application. The IDE provides tools for code editing, project management, and testing, making development more efficient and organized.
