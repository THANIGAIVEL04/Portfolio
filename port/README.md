## ðŸš€ Deployment Instructions

### GitHub Pages Deployment

1. **Create a new repository** on GitHub named `portfolio` (or any name you prefer)

2. **Initialize Git** in your local project folder:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Portfolio website"
   ```

3. **Connect to GitHub**:
   ```bash
   git remote add origin https://github.com/THANIGAIVEL04/portfolio.git
   git branch -M main
   git push -u origin main
   ```

4. **Enable GitHub Pages**:
   - Go to your repository on GitHub
   - Click on **Settings**
   - Scroll to **Pages** section
   - Under **Source**, select `main` branch
   - Click **Save**
   - Your site will be live at: `https://THANIGAIVEL04.github.io/portfolio`

### Alternative: Custom Domain

If you want to use a custom domain:
1. Add a `CNAME` file with your domain name
2. Configure DNS settings with your domain provider
3. Update GitHub Pages settings to use custom domain
